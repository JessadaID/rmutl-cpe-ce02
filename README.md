# 🎓 RMUTL CPE CE02

เว็บไซต์โปรเจกต์เพื่อการเรียนรู้และทดลองพัฒนาเว็บแอปพลิเคชันของนักศึกษาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยเทคโนโลยีราชมงคลล้านนา (RMUTL)

## 🔧 เทคโนโลยีที่ใช้

- [Svelte](https://svelte.dev/) – เฟรมเวิร์กสำหรับพัฒนาเว็บที่รวดเร็วและเบา
- [Vite](https://vitejs.dev/) – เครื่องมือสำหรับ build และพัฒนาเว็บที่เร็วมาก
- [Vercel](https://vercel.com/) – ระบบโฮสต์เว็บไซต์แบบ serverless และ CI/CD

## 🚀 เริ่มต้นใช้งาน

### 1. Clone โปรเจกต์

```bash
git clone https://github.com/JessadaID/rmutl-cpe-ce02.git
cd rmutl-cpe-ce02
```

### 2. ติดตั้ง dependency
```bash
npm install
```

### 3. รันเซิร์ฟเวอร์สำหรับพัฒนา
```bash
npm run dev
```
### 🗂️ โครงสร้างโปรเจกต์

```plaintext
rmutl-cpe-ce02/
├── src/                  # ไฟล์ต้นฉบับของแอป (components, routes, stores, etc.)
├── static/               # ไฟล์สาธารณะ เช่น รูปภาพหรือ favicon
├── package.json          # ข้อมูลและคำสั่งของโปรเจกต์
├── svelte.config.js      # การตั้งค่า Svelte
├── vite.config.js        # การตั้งค่า Vite
└── README.md             # ไฟล์คำอธิบายโปรเจกต์นี้
```
